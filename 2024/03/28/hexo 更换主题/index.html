hexo 更换主题

Created2024-03-28|Updated2024-03-28|技术
|Post Views:

假如要使用 hexo-theme-flexblock 项目，您可以按照以下步骤操作

1. 通过 Git 安装

  • 执行 git clone 命令将主题下载到您的 Hexo 项目的 themes 目录下
  • 如果您不是 Git 用户，可以从 GitHub 下载主题的压缩包，然后解压到您的 Hexo 项目目录的 themes 目录下，并重命名

2. 应用主题

  • 编辑您的 Hexo 项目目录下的 _config.yml 文件，找到 theme 配置项并设置值为 flexblock

完成以上步骤后，您就可以开始使用 hexo-theme-flexblock 主题了

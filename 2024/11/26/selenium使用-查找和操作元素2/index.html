4. 总结

通过以下方法，可以实现更复杂的元素查找：

  • 通过父元素查找子元素：通过父元素的 find_element 或 find_elements 方法查找子元素。
  • 使用 XPath：通过 XPath 可以更精确地查找元素，尤其适用于层级关系复杂的元素查找。
  • 使用 CSS 选择器：CSS 选择器灵活强大，适用于各种复杂的查找需求。
  • 兄弟元素和父元素查找：使用 XPath 的 following-sibling 和 preceding-sibling 查找兄弟元素，使用 parent::* 查找父元素。

通过这些方法，你可以在复杂的页面结构中有效地查找和操作元素。

4. 查找元素

Selenium 提供了多种方法来查找页面上的元素，例如：

  • find_element_by_id()：根据元素的 ID 查找。
  • find_element_by_name()：根据元素的名称查找。
  • find_element_by_xpath()：根据 XPath 查找元素。
  • find_element_by_css_selector()：根据 CSS 选择器查找元素。

小提示：

  • 在使用 find_element_*() 方法时，如果元素不存在，它会抛出 NoSuchElementException 异常。你可以使用 try-except 块来捕获异常。
  • 隐式等待：可以设置全局的隐式等待时间，这样每次查找元素时，如果元素还没有加载完成，会自动等待指定时间。

通过这些步骤，你可以开始使用 Selenium 自动化浏览器操作了。如果需要更复杂的操作（如滚动页面、截图等），Selenium 也能轻松实现。
